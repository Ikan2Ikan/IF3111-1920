### Cara Instalasi
  - Clone / Download Repository ke komputer local, letakkan di folder server XAMPP / WAMP / LAMP
  - Import Database, beri nama "lapor" tanpa tanda petik
  - setting base_url sesuai dengan folder server local
  - Enjoy!
